# Мониторинг ноутбуков

Веб-приложение для обнаружения и подсчета ноутбуков на изображениях с использованием модели YOLOv8.

## Возможности

- Загрузка изображения
- Автоматическое обнаружение ноутбуков
- Подсчет количества ноутбуков
- Отображение обработанного изображения
- Экспорт отчета в Excel

## Технологии

- Python
- Flask
- YOLOv8
- OpenCV
- HTML
